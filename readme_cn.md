# tio-boot

基于java aio的高性能web框架,可以让单台服务器承受上万并发  
依赖组件  

- tio-http-server
- tio-websocket-server
- jfinal-aop
- enjoy
- logback-classic
- fastjson

如果使用Java 8 开发请使用

```
<dependency>
  <groupId>com.litongjava</groupId>
  <artifactId>tio-boot</artifactId>
  <version>1.2.0</version>
</dependency>
```
